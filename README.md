# もうひとつ (Mouhitotsu)

"もう一つの重力"を操る2Dアクションパズルゲーム

## 概要

通常の下向き重力に加えて、**もう一つの重力**を獲得してゴールを目指すパズルアクションゲーム。

- 重力スイッチを踏むと追加重力が**重ねがけ**される
- 複数のスイッチを踏んで重力をカスタマイズ
- 2つの重力がベクトル合成され、独特な挙動が生まれる
- 追加重力を使って高い足場へ飛んだり、壁を登ったりしてゴールへ

## 「もうひとつ」のテーマ

このゲームでは、通常1つしか存在しないはずの**重力**がもう一つ追加されます。

## 操作方法

| キー | アクション |
|------|-----------|
| A / ← | 左移動 |
| D / → | 右移動 |
| W / ↑ / Space | ジャンプ |
| R | リトライ |

## 重力システム

### 通常重力
常に下向きに働く基本の重力 (0, -9.81)

### 追加重力（もう一つ）
ステージ上のギミックで獲得する追加の重力：

- **重力スイッチ (Add)**: 踏むと追加重力に方向を**加算**（重ねがけ）
- **重力スイッチ (Set)**: 踏むと追加重力を**上書き**
- **重力スイッチ (Clear)**: 踏むと追加重力を**リセット**

### 重ねがけシステム

スイッチを踏むたびに追加重力が加算されます：

```
例:
上スイッチ → 追加重力: ↑ (1倍)
上スイッチ → 追加重力: ↑↑ (2倍)
右スイッチ → 追加重力: 右上方向 (合成)
```

最大3回分まで重ねがけ可能（設定で変更可）

## UI

### 重力インジケーター
画面左上に現在の重力状態を表示：
- 通常重力（グレー矢印）
- 追加重力（シアン矢印）
- 合成重力（オレンジ矢印）
- 追加重力の強さ（←←← のように矢印の数で表示）
