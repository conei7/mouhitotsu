# もうひとつ (Mouhitotsu)

2つの世界が重なる横スクロールパズルアクションゲーム

## 概要

青と赤の2体のキャラクターを**同時に操作**し、両方を**同時にゴール**させることが目的のパズルアクションゲーム。

- 青キャラは青い壁にのみ衝突
- 赤キャラは赤い壁にのみ衝突
- 色付きの壁を使って片方を止め、もう片方を誘導するパズル

## 「もうひとつ」のテーマ

このゲームでは、通常1つしか存在しないはずの**カメラ**が2つの世界を同時に映し出します。

## 操作方法

| キー | アクション |
|------|-----------|
| A / ← | 左移動 |
| D / → | 右移動 |
| W / ↑ / Space | ジャンプ |
| R | リトライ |
