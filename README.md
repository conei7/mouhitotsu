# もうひとつ (Mouhitotsu)

"もう一つの重力"を操る2Dアクションパズルゲーム

## 概要

通常の下向き重力に加えて、**もう一つの重力**を獲得してゴールを目指すパズルアクションゲーム。

- 重力スイッチを踏むと追加重力の方向が変わる
- 2つの重力がベクトル合成され、独特な挙動が生まれる
- 追加重力を使って高い足場へ飛んだり、壁を登ったりしてゴールへ

## 「もうひとつ」のテーマ

このゲームでは、通常1つしか存在しないはずの**重力**がもう一つ追加されます。

## 操作方法

| キー | アクション |
|------|-----------|
| A / ← | 左移動 |
| D / → | 右移動 |
| W / ↑ / Space | ジャンプ |
| R | リトライ |

## 重力システム

### 通常重力
常に下向きに働く基本の重力

### 追加重力（もう一つ）
ステージ上のギミックで獲得する追加の重力：
- **重力スイッチ**: 踏むと追加重力の方向が切り替わる
- **重力オーブ**: 拾うと一時的に追加重力を獲得

追加重力は通常重力の**0.5倍の強さ**で合成されます。
